Introduction

What is HTML?

HTML es un lenguaje de marcado de hipertexto, es decir, es el código que se utiliza para crear y darle estructura a una página web, es como el esqueleto que le da estructura al cuerpo humano

Explain the head and the body tags.

La etiqueta head es un contenedor de datos es decir almacena datos relevantes de la página web como el nombre del sitio, estilos, etc.

La etiqueta body es donde se almacena el contenido que queremos mostrar en la página web, puede incluir varias características para definir algunos aspectos de la apariencia de la página.

HTML - Basic Tags

Explain the section, article, aside, header, footer tag?

Piensa en una página web como si fuera un periodico o una revista
header: Sería como el título grandote de arriba de la revista con el logo.
footer: Serían las letras chiquitas de hasta abajo de la revista con los derechos de autor y el contacto.
article: Sería como una noticia completa o independiente. Si se recorta y se pega en otra revista, sigue teniendo sentido.
section: Esta estiqueta vendría siendo una categoría o sección de la revista (ej. "Sección de Deportes"), que agrupa varias cosas juntas.
aside: Esta etiqueta es la barra lateral que tiene anuncios o información "extra" (como chismes) que no es la historia principal.

2. CSS - Introduction

What is CSS? & What is the purpose of CSS?

Si HTML es el esqueleto, CSS vendría siendo la piel, el que le da diseño a las páginas web, para que se vean bonitas, limpias y su contenido sea fácil de leer. Su propósito es darle diseño y estilo a una página web.

How do you apply a CSS to an HTML (explain the 3 options)

Tienes 3 formas de pintar la casa:
Inline (En línea): Agarras una brocha y pintas directamente sobre un solo ladrillo (se escribe el código directo en la etiqueta HTML). Muy tedioso.
Internal (Interno): Pones una cubeta de pintura en la sala y desde ahí pintas toda la casa (se pone una etiqueta `style` dentro del `head` del HTML).
External (Externo): Contratas a un pintor profesional que trae su propio manual y herramientas desde afuera (creas un archivo `.css` separado y lo linkeas al HTML).
Reference: W3Schools CSS How-To.

What is a selector? What is a CSS rule?

Un selector es tú apuntando con el dedo y diciendo "A esa silla de ahí". Una regla (CSS rule) es la orden que le das al pintor, diciendo "Píntala de rojo y ponle brillos".
Reference: MDN Web Docs - CSS Selectors.

CSS - Basic CSS

Explain a: Tag Selector, Class Selector, Tag and Class Selector, Id Selector

IMAGINA QUE ESTAS EN UN CLUB O UN EVENTO
Tag Selector: Es como decir "Todos los que son estudiantes hagan esto".
Class Selector: Es como ponerle una pulsera especial a ciertas personas en el evento “Todos los que tengan pulsera VIP hacen esto”.
Tag and Class Selector: Ahora es más específico Ej: "Solo los estudiantes que tienen pulsera VIP hacen esto".
Id Selector: Es como decir: “Juan Pérez específicamente haz esto”.
Reference: MDN Web Docs - CSS Selectors basics.

What is a class? What is an id? How are they related to CSS?

Una class (clase) es como el uniforme de un equipo de fútbol; muchos jugadores pueden usar la misma camisa para verse iguales. Un id es como tu número de pasaporte o tu huella digital; solo TÚ puedes tenerlo en toda la página. En CSS, los usas para saber exactamente a quién le vas a cambiar el estilo sin equivocarte de persona.
Reference: W3Schools HTML classes and id attributes.

Explain the CSS Box Model? Show an example.

Contenido (content): Es lo que hay dentro (el regalo).

Explain the media queries

Son como unos lentes mágicos para la página web. Las media queries le dicen a la página web: "Hey, si me estás viendo desde un celular chiquito, acomoda todo en una sola fila. Pero si me estás viendo desde una televisión gigante, pon todo en tres columnas". Adapta el diseño dependiendo del tamaño de la pantalla.
Reference: MDN Web Docs - CSS Media Queries.

CSS - Layout CSS

Explain display: grid

El Grid es como excel o ajedrez. Te permite crear una cuadrícula perfecta con filas (rows) y columnas (columns) en 2D. Tú decides exactamente en qué "cuadrito" del tablero de ajedrez quieres poner cada elemento de tu página web.

Explain display: flex

Flexbox es como una fila. Es para organizar cosas en una sola dirección (ya sea en una fila horizontal o en una columna vertical). Es súper flexible (de ahí el nombre) porque los elementos se pueden estirar o encoger automáticamente para llenar los espacios vacíos.
