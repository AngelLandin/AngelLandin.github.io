¿Puedo ver la imagen?

Respuesta: No, la imagen aparece rota. Esto sucede porque el servidor de Node.js actual (servidor.js) solo tiene rutas definidas con la lógica de if/else para devolver HTML o JSON. No hemos programado ninguna función para leer y servir archivos estáticos con extensiones como .jpg o .png. Por lo tanto, cuando el navegador pide la imagen, el servidor cae en la ruta por defecto y devuelve el error 404.
